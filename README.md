 Auto Message Sender (CLI Bot)

A simple Python CLI bot that automatically sends messages based on user input.

 Features
Send custom messages
Choose how many times to send
Set delay between messages
Preview message before sending
Edit options before execution
Simple and interactive terminal interface
 How It Works

The user selects an option from the menu and provides:

Message content
Number of repetitions
Target (username)
Delay between messages

The bot then shows a preview and asks for confirmation before sending.

 Usage
Run the script:
python main.py
Follow the instructions in the terminal:
Enter message
Set number of times
Choose delay
Confirm execution
🛠 Tech Used
Python
time module (for delays)

 Example
====  This is a bot for automatic message sending  ====

Please choose an option:
1. Send Message
2. Exit

> 1

Enter message: Hello
Times: 3
Delay: 2

Preview:
Message: Hello
Times: 3
Delay: 2s
